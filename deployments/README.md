# 環境作成用のコード

## 動かすための前提条件

以下のコマンドがインストールされていること

### `awscli`

Mac で `brew` を使っている場合、以下のコマンドでインストール可能。

```bash
brew install awscli
```

### `awscdk`

Mac で `brew` を使っている場合、以下のコマンドでインストール可能。

```bash
brew install aws-cdk
```

`CDK` の使い方については、 [CDK README](./README.cdk.md) を参照
